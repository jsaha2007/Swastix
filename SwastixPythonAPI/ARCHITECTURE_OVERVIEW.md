## ?? Authentication Flow

```
User
 ?
 ?? Enter Email/Phone
 ?
 ?? System generates OTP
 ?
 ?? OTP sent via SMS/Email
 ?
 ?? User enters OTP
 ?
 ?? System validates OTP
 ?
 ?? If valid: Create/Update User
 ?
 ?? Return Auth Token (JWT)
    ?
    ?? User logged in ?
```

## ?? Getting Started

### 1. Prerequisites
```
- Python 3.8+
- PostgreSQL 12+
- pip (Python package manager)
```

### 2. Installation
```bash
cd SwastixPythonAPI
python -m venv .venv
.venv\Scripts\activate.bat  # Windows
pip install -r requirements.txt
```

### 3. Database Setup
```bash
# Create database
createdb swastix_db

# Initialize tables
python init_db.py
```

### 4. Verify
```bash
python -c "from models import *; print('? Ready!')"
```
